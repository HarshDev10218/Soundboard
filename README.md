# 🎵 Mini Soundboard

A simple retro-themed soundboard built with **HTML**, **CSS**, and **JavaScript**. Click any button to play a sound effect, adjust the volume with the slider, or stop all currently playing sounds.

## ✨ Features

- 🎶 12 built-in sound effects
- 🔊 Real-time volume control
- 🛑 Stop all playing sounds
- 📱 Responsive grid layout
- 🎨 Retro-inspired dark UI
- 🚀 Lightweight with no external libraries

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/mini-soundboard.git
```

2. Open the project folder.

3. Ensure all sound files are placed inside the `sounds/` directory.

4. Run the project using a local server.

### VS Code Live Server

- Install the **Live Server** extension.
- Right-click `index.html`.
- Select **Open with Live Server**.

> Running through a local server is recommended because some browsers restrict audio playback when opening HTML files directly.

## 🎮 How to Use

- Click any sound button to play its sound effect.
- Adjust the **Volume** slider to control playback volume.
- Click **Stop All** to immediately stop every playing sound.
